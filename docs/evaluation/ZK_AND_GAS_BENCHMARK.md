## Boundary Statement

The circuit public inputs are fixed to:

```text
[tally[0], tally[1], tally[2], tally[3], batchSize]
```

The Solidity verifier proves arithmetic tally correctness for those public
signals. The API binds `electionIdHash`, `tallyHash`, `commitmentRoot`,
`partitionHash`, `proofHash`, and verifier mode against the current
AggregatorReport before calling the contract. `MockTallyVerifier` and
`local-mock` are fallback/demo paths only and must not be described as real
on-chain Groth16 verification.
